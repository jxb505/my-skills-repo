# 安全生产违章分析框架

## 输出结构
1. 报告封面/摘要：数据周期、有效记录数、覆盖单位、关键发现。
2. 总体画像：违章总量、趋势、级别、分值、人员性质、数据来源。
3. 突出问题：从类别、区域、单位、作业类型、人员性质、检查层级中识别系统性短板。
4. 高频/重复性违章：列出 TOP 类别、TOP 区域、TOP 责任/协力单位、典型条款。
5. 管理履职分析：检查人员类别、数据来源、连带责任、甲方/协力管理缺口。
6. 根因诊断：制度、培训、现场监督、承包商管理、技术防御、行为文化六类原因。
7. 改进建议：按短期专项整治、中期体系优化、长期安全文化建设给出行动清单。
8. 附录：关键统计表、图表说明、数据口径。

## 常用图表
- 月度/周度趋势折线图：识别波峰和治理窗口。
- 违章级别占比饼图或环形图：突出 A/B/C 类风险结构。
- 违章类别 TOP10 条形图：定位高频问题。
- 组织/区域 TOP10 条形图：定位重点责任区域。
- 人员性质占比图：比较正式员工、协力人员、外来人员风险。
- 数据来源占比图：判断线下巡查与视频行为观察占比。
- 级别 x 类别透视热力图：识别高频且高风险问题。

## 分析原则
- 先说明数据口径：是否剔除已删除、是否不含连带、统计周期。
- 不只报数，要解释数字背后的管理含义。
- 对 A 类、高分、重复出现、协力单位集中、视频回看发现占比高等现象提高权重。
- 建议必须可执行：明确责任对象、行动方式、衡量指标和周期。
